INTRODUCTION
------------
This is an Eclipse project for an AWS serverless web service. The web service has one PUT handler and two GET handlers, which store an id, title, and info in a DynamoDB table.

SETUP
-----
Install Eclipse and add the AWS Toolkit for Eclipse. Download this repository and import into Eclipse.

REQUIREMENTS
------------
 * AWS account
 * Knowledge of AWS services: S3, DynamoDB, API Gateway, and serverless projects
 * Eclipse w/ AWS Toolkit for Eclipse
